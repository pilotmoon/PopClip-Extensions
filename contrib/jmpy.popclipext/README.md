# Jmpy PopClip Extension

A custom PopClip extension for macOS users to quickly shorten URLs and generate dynamic tracking QR codes using Jmpy.

## Features
1. **Jmpy Short URL**: Shorten the selected URL using Jmpy.me and copy the short link directly to your clipboard.
2. **Jmpy QR Code**: Generate a dynamic QR code for the selected URL and open the download page in your default browser.

## How to Install (Manually)
To install this extension manually:
1. Download the `jmpy.popclipext` folder.
2. Double-click the folder in Finder. PopClip will prompt you to install.

## Configuration
This extension requires a **Jmpy API Key** to function:
1. Create a free account at [https://jmpy.me](https://jmpy.me).
2. Go to your User Dashboard -> **API Settings**.
3. Generate a new API Key (which starts with `jmpy_`).
4. Click the Jmpy extension in PopClip settings and enter your API Key.

# Slack mrkdwn

Convert between Slack mrkdwn and standard Markdown/rich text. Two actions:

**Copy as Slack mrkdwn** — Converts rich text (HTML) or plain Markdown to
Slack's mrkdwn format. Handles bold (`*`), italic (`_`), strikethrough (`~`),
links (`<url|text>`), headings (converted to bold), and bullet lists (`•`).

**Slack to Markdown** — Converts Slack mrkdwn text to standard Markdown.
Reverses the Slack-specific formatting back to `**bold**`, `~~strike~~`,
`[text](url)`, etc.

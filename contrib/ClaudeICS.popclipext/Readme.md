# Claude ICS

Generate ICS calendar files from selected text using Claude AI.

## Usage

1. Select text containing event details (e.g., "Meeting with the team tomorrow at 2pm")
2. Click the Claude ICS extension in PopClip
3. Claude will open with a prompt to generate an ICS calendar file from your selected text
4. The extension automatically includes today's date for context and assumes future dates when no year is specified

## Requirements

- Claude desktop app must be installed
- Available at: https://claude.ai

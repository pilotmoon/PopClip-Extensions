# BBEdit

Send the text to a new document in BBEdit.

## About

This is an extension for [PopClip](https://www.popclip.app/).

### Author

Nick Moore, [Javier JJ](https://github.com/jjarava)

### Requirements

Requires PopClip 2021.11

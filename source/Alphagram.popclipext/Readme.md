# Alphagram

Creates an alphagram from each word of the selected text, or scrambles them
randomly, for anagram solving and classroom games.

- **Alphagram**: Rearranges the letters of each selected word into alphabetical
  order and converts them to uppercase

- **Scramble**: Randomly shuffles the letters of each selected word and converts
  them to uppercase.

The extension transforms letters only. Digits, spaces, punctuation, and line
breaks stay in place unchanged.

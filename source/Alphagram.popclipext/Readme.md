# Alphagram

Creates an alphagram from the selected text, or scrambles it randomly. This is
useful for anagram solving and classroom games.

- **Alphagram**: Rearranges the selected letters into alphabetical order and
  converts them to uppercase.

- **Scramble**: Randomly shuffles the selected letters and converts them to
  uppercase.

The extension transforms letters only. Digits, spaces, punctuation, and line
breaks stay in place unchanged.
